gem5安装配置

由于实验需要，这些天尝试安装了gem5到虚拟机中，以下为安装记录。

实验环境：WMware + Ubuntu-14.04.3-desktop-amd64。

虚拟机配置

1.在WMware中安装Ubuntu-14.04.3

2.配置Ubuntu root账号

3.安装 WMware tool

root下安装gem5

1. 安装Mercurial版本控制软件，用于获取gem5代码

2. 更新编译环境

3. 安装scon工具

4. 安装python

5. 安装swig

6. 安装zlib

7. 安装automake

8. 获得gem5 代码

构建gem5

1. 构建gem5，构建出的系统为build/X86/gem5.opt

编译选项文件存放在 build_opts，本次安装没有指定编译选项文件，且编译结果存放在build/x86文件夹下，因此scon将在build_opts下自动寻找x86的编译选项文件。过程持续较长时间。
